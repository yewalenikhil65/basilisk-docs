# Basilisk CFD — API Reference {#mainpage}

[Basilisk](http://basilisk.fr) is a free-software framework for
solving PDEs on adaptive Cartesian meshes (quadtree/octree).

This is the **machine-generated API reference**. For the human-readable
literate documentation (prose + code as the authors wrote it), see the
companion [Literate Documentation](../literate/index.html) site.

## What this reference provides

- **Data Structures** — Grid, scalar, vector, tensor, coord, _Attributes, mgstats
- **Collaboration Diagrams** — struct composition relationships
- **Include Graphs** — which headers depend on which (click any file page)
- **Call / Caller Graphs** — function-level dependency chains
- **Cross-referenced Source** — every identifier links to its definition
- **Directory Graphs** — module-level dependency structure
- **Full-text Search** — find any symbol instantly
